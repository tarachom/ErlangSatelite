
Прийом вхідних підключень по протоколу TCP від Erlang

-------------------------------------------------------------

Програма для тестування взаємодії Erlang та програми на C# по протоколу TCP/IP

На Erlang запущений веб-сервер httpd.
Програма ErlangSatelite.exe відкриває 5555 порт і очікує підключення.

Скріпт http://127.0.0.1/erl/scripts/client підключається до програми ErlangSatelite.exe, 
передає параметри і отримує відповідь яку пересилає браузеру.

БРАУЗЕР (http://127.0.0.1/erl/scripts/client) <-- --> Erlang <-- --> ErlangSatelite.exe
